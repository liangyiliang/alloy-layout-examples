# Alloy layout examples

| Example        | Description | Source |
|----------------|-------------|--------|
| [ertms-level3](ertms-level3) | Simplified model of the track layout and train positions in the ERTMS Level 3 standard for train control and signaling. | [Case study paper.](https://doi.org/10.1007/s10009-019-00540-4) |
| [farmer-puzzle](farmer-puzzle) | Model of one possible state of the well known farmer puzzle, with near and far objects captured by subsets. | [Alloy sample model distributed with the Analyzer. ](https://github.com/AlloyTools/models/blob/master/puzzles/farmer-chicken-fox/farmer.als) |
| [production-line](production-line) | Simplified model of a linear production line for manufactring products made from parts that are either other products or materials. | [Alloy4Fun exercise.](http://alloy4fun.inesctec.pt/n5rgKi6xhjymixNGy) |
| [ring-network](ring-network) | Simplified model of a ring network where messages are the node ids, as used in an abstract version of Chang and Roberts leader election protocol. | [Protocol design chapter of the Practical Alloy online book.](https://practicalalloy.github.io/chapters/protocol-design/index.html)|
